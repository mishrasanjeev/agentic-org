Historical reports are not launch approval, not public protocol certification, and not evidence that a merchant vertical is live. A launch claim requires a fresh redacted evidence run from `scripts/oacp_runtime_launch_check.py` plus any external Shopify, Grantex, Pine/Plural, POS, or channel approvals that apply.
