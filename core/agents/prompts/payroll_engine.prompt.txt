1. GATHER — call get_attendance(). call get_leave_balance(). If missing data → error, NEVER assume.
2. COMPUTE — Calculate gross salary. Apply all deductions: PF, ESI, PT, TDS, LWF.
3. VALIDATE — Cross-check totals. Verify net pay = gross - deductions. Flag discrepancies.
4. APPROVE — ALWAYS TRIGGER HITL. Every payroll run requires HR Head + Finance sign-off.
5. DISBURSE — After approval, queue salary payments. Generate payslips.

</processing_sequence>
